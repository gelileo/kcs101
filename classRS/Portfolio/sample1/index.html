Programming 101 For Kids

An introductory course to software programming for beginners

About This Course

Have you ever wished you knew how to program, but had no idea where to start from? Or are you bored with learning code on Scratch and wondering what a real program looks like?

This course will introduce you to realworld pragramming lauguages. More importantly, it will introduce you to the fundamental coding concepts and help you think like a programmer.

Programming Concepts

  • Basic Data Types and Structures
  • Baisc Flow Controls

Programming Languages/Skills

  • Intro to JavaScript
  • Web Page with HTML and CSS
  • Interactive Web With JavaScript
  • HTML Canvas Game Development

Prerequisites

  • No previous programming knowledge required
  • Recommended Student Age: 10 - 15
  • A Gmail account
  • A Laptop/MacBook/iPad/Chromebook with WiFi

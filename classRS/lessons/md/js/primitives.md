
## Primitive Types

  - numbers
    - [Numbers](./numbers.md)
  - strings
    - [Strings](./strings.md)
    
  - booleans
    `true` / `false`
    ```
      console.log("hello" === "hello") // true
    ```

    A value of this type can be created using Symbol():
    ```
      let id = Symbol()
    ```

 ## Calling a JS Function
`function-name([parameters])`
 
```
  brick(0, 0, 80, 20);
  
  text("Hello", canvas.width / 2, 150, 'white', ' 1em Arial', 'center');

  circle(canvas.width / 2, 200, 20, 'yellow');
 ```
